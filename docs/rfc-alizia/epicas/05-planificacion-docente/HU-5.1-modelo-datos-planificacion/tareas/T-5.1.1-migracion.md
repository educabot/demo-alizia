# T-5.1.1: Migracion — tablas de planificacion

**Historia:** HU-5.1 — Modelo de datos planificacion
**Tipo:** DB
**Estado:** ⬜ Pendiente

---

## Descripcion

Crear las tablas `teacher_lesson_plans` y `lesson_plan_activities` con sus enums y constraints.

## Implementacion

```sql
-- Enum para estado de planificacion
CREATE TYPE lesson_plan_status AS ENUM ('pending', 'planned');

-- Enum para tipo de fuente (nullable)
CREATE TYPE source_type AS ENUM ('resource', 'custom');

CREATE TABLE teacher_lesson_plans (
    id SERIAL PRIMARY KEY,
    organization_id INTEGER NOT NULL REFERENCES organizations(id),
    course_subject_id INTEGER NOT NULL REFERENCES course_subjects(id),
    coordination_document_id INTEGER NOT NULL REFERENCES coordination_documents(id),
    coord_doc_class_id INTEGER NOT NULL REFERENCES coord_doc_classes(id),
    class_number INTEGER NOT NULL,
    title VARCHAR(500) NOT NULL,
    objective TEXT,
    status lesson_plan_status NOT NULL DEFAULT 'pending',
    proposal TEXT,
    source_type source_type,
    source_reference TEXT,
    created_at TIMESTAMP NOT NULL DEFAULT NOW(),
    updated_at TIMESTAMP NOT NULL DEFAULT NOW(),
    UNIQUE(course_subject_id, coord_doc_class_id)
);

CREATE TABLE lesson_plan_activities (
    id SERIAL PRIMARY KEY,
    teacher_lesson_plan_id INTEGER NOT NULL REFERENCES teacher_lesson_plans(id) ON DELETE CASCADE,
    moment class_moment NOT NULL,
    activity_id INTEGER NOT NULL REFERENCES activities(id),
    custom_note TEXT,
    "order" INTEGER NOT NULL DEFAULT 0,
    UNIQUE(teacher_lesson_plan_id, moment, activity_id)
);

CREATE INDEX idx_lesson_plans_course_subject ON teacher_lesson_plans(course_subject_id);
CREATE INDEX idx_lesson_plans_coord_doc ON teacher_lesson_plans(coordination_document_id);
CREATE INDEX idx_lesson_plan_activities_plan ON lesson_plan_activities(teacher_lesson_plan_id);
```

## Notas

- Reutiliza el enum `class_moment` creado en HU-3.6 (opening, development, closing)
- `source_type` es nullable: null = sin fuente anclada
- `proposal` es nullable: se llena al generar con IA
- ON DELETE CASCADE en activities para que al borrar un plan se borren sus actividades
