# T-5.2.1: Usecase -- listar clases del docente

**Historia:** HU-5.2 -- Cronograma del docente
**Tipo:** Backend
**Estado:** ⬜ Pendiente

---

## Descripcion

Usecase que combina las clases del documento de coordinacion con las planificaciones existentes del docente y la informacion de clases compartidas.

## Implementacion

```go
// src/core/usecases/get_teacher_schedule.go

type GetTeacherScheduleUsecase struct {
    courseSubjectProvider       providers.CourseSubjectProvider
    coordDocProvider           providers.CoordinationDocumentProvider
    lessonPlanProvider         providers.TeacherLessonPlanProvider
    sharedClassProvider        providers.SharedClassProvider
}

type TeacherClassView struct {
    ClassNumber       int                          `json:"class_number"`
    CoordDocClassID   uint                         `json:"coord_doc_class_id"`
    Title             string                       `json:"title"`
    Objective         *string                      `json:"objective"`
    IsShared          bool                         `json:"is_shared"`
    SharedWithSubject *string                      `json:"shared_with_subject"`
    PlanStatus        *entities.LessonPlanStatus   `json:"plan_status"`
    LessonPlanID      *uint                        `json:"lesson_plan_id"`
}

func (uc *GetTeacherScheduleUsecase) Execute(courseSubjectID uint) (*TeacherScheduleResponse, error) {
    // 1. Get course_subject with preloads
    cs, err := uc.courseSubjectProvider.GetByID(courseSubjectID)

    // 2. Find published coordination_document for this area
    doc, err := uc.coordDocProvider.GetPublishedByArea(cs.Subject.AreaID)
    if doc == nil {
        return nil, errors.NewNotFound("no published coordination document")
    }

    // 3. Get coord_doc_classes for this subject
    classes := getClassesForSubject(doc, cs.SubjectID)

    // 4. Get existing lesson plans (map by coord_doc_class_id)
    plans, _ := uc.lessonPlanProvider.GetByCourseSubject(courseSubjectID)
    planMap := buildPlanMap(plans) // map[uint]*TeacherLessonPlan

    // 5. Get shared class numbers
    sharedNumbers, _ := uc.sharedClassProvider.CalculateSharedClassNumbers(courseSubjectID)

    // 6. Build combined view
    var result []TeacherClassView
    for _, class := range classes {
        view := TeacherClassView{
            ClassNumber:     class.ClassNumber,
            CoordDocClassID: class.ID,
            Title:           class.Title,
            Objective:       &class.Objective,
            IsShared:        sharedNumbers.Contains(class.ClassNumber),
        }
        if plan, ok := planMap[class.ID]; ok {
            view.PlanStatus = &plan.Status
            view.LessonPlanID = &plan.ID
        }
        result = append(result, view)
    }
    return &TeacherScheduleResponse{Classes: result}, nil
}
```

## Notas

- El usecase no crea planificaciones automaticamente -- solo muestra el estado actual
- `plan_status: null` indica que el docente aun no empezo a planificar esa clase
- La info de shared_with_subject se obtiene del calculo de clases compartidas (HU-3.5)
