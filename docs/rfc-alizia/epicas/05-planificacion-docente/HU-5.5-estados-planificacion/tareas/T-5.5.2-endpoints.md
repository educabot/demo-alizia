# T-5.5.2: Endpoints de estado y vista coordinador

**Historia:** HU-5.5 — Estados de planificación
**Tipo:** Backend
**Estado:** ⬜ Pendiente

---

## Descripción

Endpoint para cambiar estado del plan y endpoint para vista de progreso del coordinador.

## Implementación

### Endpoints

```
PATCH /api/v1/lesson-plans/:id/status              → Cambiar estado
GET   /api/v1/coordination-documents/:id/planning-progress → Progreso por materia
```

### Handler cambio de estado

```go
func (h *LessonPlanHandler) ChangeStatus(c *gin.Context) {
    planID := parseID(c, "id")
    var input usecases.ChangeLessonPlanStatusInput
    if err := c.ShouldBindJSON(&input); err != nil {
        c.JSON(400, gin.H{"error": err.Error()})
        return
    }
    user := auth.GetUser(c)
    plan, err := h.changeStatusUsecase.Execute(user.ID, planID, input)
    if err != nil {
        handleError(c, err)
        return
    }
    c.JSON(200, plan)
}
```

### Handler progreso coordinador

```go
func (h *LessonPlanHandler) GetPlanningProgress(c *gin.Context) {
    docID := parseID(c, "id")

    // Auth: solo coordinador del área
    user := auth.GetUser(c)
    doc, _ := h.coordDocProvider.GetByID(docID)
    if !isCoordinatorOfArea(user, doc.AreaID) {
        c.JSON(403, gin.H{"error": "not authorized"})
        return
    }

    progress, err := h.getPlanningProgressUsecase.Execute(docID)
    if err != nil {
        handleError(c, err)
        return
    }
    c.JSON(200, progress)
}
```

### Query progreso

```sql
SELECT
    cds.subject_id,
    s.name as subject_name,
    u.name as teacher_name,
    (SELECT COUNT(*) FROM coord_doc_classes cdc WHERE cdc.coordination_document_subject_id = cds.id) as total_classes,
    (SELECT COUNT(*) FROM teacher_lesson_plans tlp
     WHERE tlp.coordination_document_id = $1
       AND tlp.course_subject_id = cs.id
       AND tlp.status = 'planned') as planned_classes,
    (SELECT COUNT(*) FROM teacher_lesson_plans tlp
     WHERE tlp.coordination_document_id = $1
       AND tlp.course_subject_id = cs.id
       AND tlp.status = 'pending') as pending_classes
FROM coordination_document_subjects cds
JOIN subjects s ON s.id = cds.subject_id
JOIN course_subjects cs ON cs.subject_id = cds.subject_id
JOIN users u ON u.id = cs.teacher_id
WHERE cds.coordination_document_id = $1;
```

## Notas

- `not_started = total_classes - planned_classes - pending_classes`
- `progress_percentage = (planned_classes * 100) / total_classes`
