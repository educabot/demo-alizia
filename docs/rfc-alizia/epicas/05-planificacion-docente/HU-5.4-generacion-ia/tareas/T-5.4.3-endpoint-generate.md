# T-5.4.3: Endpoint POST generate

**Historia:** HU-5.4 — Generación con IA
**Tipo:** Backend
**Estado:** ⬜ Pendiente

---

## Descripción

Endpoint para generar la propuesta de clase con IA.

## Implementación

### Endpoint

```
POST /api/v1/lesson-plans/:id/generate
```

### Handler

```go
func (h *LessonPlanHandler) Generate(c *gin.Context) {
    planID := parseID(c, "id")
    user := auth.GetUser(c)

    plan, err := h.generateUsecase.Execute(user.ID, planID)
    if err != nil {
        handleError(c, err)
        return
    }
    c.JSON(200, plan)
}
```

### Response

Retorna el plan completo con el campo `proposal` actualizado.

## Notas

- No requiere body — toda la info se toma del plan existente
- 200 (no 201) porque actualiza un recurso existente
