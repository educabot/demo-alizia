# T-5.4.1: Prompt y schema para generación

**Historia:** HU-5.4 — Generación con IA
**Tipo:** Backend
**Estado:** ⬜ Pendiente

---

## Descripción

Definir el prompt del sistema y la estructura de contexto para generar propuestas de clase.

## Implementación

### System prompt

```
Sos Alizia, una asistente pedagógica que ayuda a docentes a planificar sus clases.

Tu tarea es generar una propuesta detallada de clase basada en:
- El título y objetivo de la clase
- Las actividades seleccionadas por el docente para cada momento
- La fuente o recurso anclado (si existe)
- Los temas/categorías curriculares de esta clase
- La estrategia metodológica del documento de coordinación

Reglas:
- Escribí en español rioplatense (vos, usá, hacé)
- Estructura la propuesta por momentos: Apertura, Desarrollo, Cierre
- Incluí tiempos estimados para cada actividad
- Sé específica: describí qué hace el docente, qué hacen los alumnos, qué materiales se necesitan
- Si hay una fuente anclada, integrala naturalmente en la propuesta
- Si es una clase compartida, mencioná cómo se articula con la otra materia
- No repitas textualmente el objetivo, integralo en la narrativa
```

### Context builder

```go
type LessonPlanGenerationContext struct {
    ClassNumber         int                `json:"class_number"`
    Title               string             `json:"title"`
    Objective           string             `json:"objective"`
    Topics              []string           `json:"topics"`
    Activities          []ActivityContext   `json:"activities"`
    Source              *SourceContext      `json:"source,omitempty"`
    MethodologicalStrategy string          `json:"methodological_strategy"`
    IsShared            bool               `json:"is_shared"`
    SharedWithSubject   *string            `json:"shared_with_subject,omitempty"`
    SharedSubjectTopics []string           `json:"shared_subject_topics,omitempty"`
}

type ActivityContext struct {
    Moment      string `json:"moment"`
    Name        string `json:"name"`
    Description string `json:"description"`
    Duration    int    `json:"duration_minutes"`
    CustomNote  string `json:"custom_note,omitempty"`
}

type SourceContext struct {
    Type      string `json:"type"`
    Reference string `json:"reference"`
}
```

## Notas

- El prompt usa español rioplatense (voseo) alineado con el contexto argentino del proyecto
- Para clases compartidas, se incluyen los topics de la otra materia para facilitar articulación
- Los tiempos de cada actividad vienen de la tabla activities.duration_minutes
