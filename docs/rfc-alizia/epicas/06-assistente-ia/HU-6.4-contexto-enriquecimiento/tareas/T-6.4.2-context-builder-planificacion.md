# T-6.4.2: Context builder planificacion

**Historia:** HU-6.4 — Contexto y enriquecimiento
**Tipo:** Backend
**Estado:** ⬜ Pendiente

---

## Descripcion

Implementar el context builder para planificaciones docentes que transforma la entidad `TeacherLessonPlan` y datos relacionados en texto enriquecido con formato markdown. Incluye info de la clase, actividades agrupadas por momento, fuente anclada, estrategia del documento de coordinacion y contexto de clase compartida.

## Implementacion

```go
// src/core/usecases/ai/context_lesson_plan.go
package ai

import (
    "fmt"
    "strings"

    "go-alizia/src/core/entities"
)

// LessonPlanContextInput aggregates all data needed to build the lesson plan context.
type LessonPlanContextInput struct {
    Plan          *entities.TeacherLessonPlan
    CoordDocClass *entities.CoordDocClass         // title, objective, topics
    CoordDoc      *entities.CoordinationDocument   // strategy, sections
    SharedWith    *entities.Subject                // nil if not shared
    SharedTopics  []string                         // topics of the other subject (if shared)
}

// BuildLessonPlanContext creates a rich text context from a lesson plan
// for use in AI proposal generation (HU-5.4).
func BuildLessonPlanContext(input LessonPlanContextInput) string {
    var parts []string

    // Clase
    parts = append(parts, fmt.Sprintf("## Clase %d: %s\nObjetivo: %s",
        input.CoordDocClass.ClassNumber,
        input.CoordDocClass.Title,
        input.CoordDocClass.Objective))

    // Topics de la clase
    if len(input.CoordDocClass.Topics) > 0 {
        topicNames := make([]string, len(input.CoordDocClass.Topics))
        for i, t := range input.CoordDocClass.Topics {
            topicNames[i] = t.Name
        }
        parts = append(parts, fmt.Sprintf("## Topics\n%s", strings.Join(topicNames, ", ")))
    }

    // Actividades por momento
    if len(input.Plan.Activities) > 0 {
        grouped := groupByMoment(input.Plan.Activities)
        var actParts []string
        for _, moment := range []string{"apertura", "desarrollo", "cierre"} {
            if acts, ok := grouped[moment]; ok {
                var lines []string
                for _, a := range acts {
                    line := fmt.Sprintf("- %s (%d min)", a.Activity.Name, a.Activity.DurationMinutes)
                    if a.TeacherNote != "" {
                        line += fmt.Sprintf(" — Nota del docente: %s", a.TeacherNote)
                    }
                    lines = append(lines, line)
                }
                actParts = append(actParts, fmt.Sprintf("### %s\n%s", strings.Title(moment), strings.Join(lines, "\n")))
            }
        }
        parts = append(parts, fmt.Sprintf("## Actividades seleccionadas\n%s", strings.Join(actParts, "\n\n")))
    }

    // Fuente anclada
    if input.Plan.SourceType != "" {
        switch input.Plan.SourceType {
        case "resource":
            parts = append(parts, fmt.Sprintf("## Recurso anclado\nURL: %s", input.Plan.SourceValue))
        case "custom":
            parts = append(parts, fmt.Sprintf("## Nota personalizada del docente\n%s", input.Plan.SourceValue))
        }
    }

    // Estrategia metodologica del documento de coordinacion
    if input.CoordDoc != nil {
        sections := input.CoordDoc.ParseSections()
        if strategy, ok := sections["methodological_strategy"]; ok {
            if v, ok := strategy["value"].(string); ok && v != "" {
                parts = append(parts, fmt.Sprintf("## Estrategia metodologica\n%s", v))
            }
        }
    }

    // Clase compartida
    if input.SharedWith != nil {
        shared := fmt.Sprintf("## Clase compartida\nEsta clase se dicta en simultaneo con **%s**.", input.SharedWith.Name)
        if len(input.SharedTopics) > 0 {
            shared += fmt.Sprintf("\nTopics de %s en esta clase: %s", input.SharedWith.Name, strings.Join(input.SharedTopics, ", "))
        }
        parts = append(parts, shared)
    }

    return strings.Join(parts, "\n\n")
}

func groupByMoment(activities []entities.LessonPlanActivity) map[string][]entities.LessonPlanActivity {
    grouped := make(map[string][]entities.LessonPlanActivity)
    for _, a := range activities {
        grouped[string(a.Activity.Moment)] = append(grouped[string(a.Activity.Moment)], a)
    }
    return grouped
}
```

## Comportamiento

1. Recibe un `LessonPlanContextInput` con todos los datos pre-cargados por el usecase
2. Construye las secciones en orden fijo: clase → topics → actividades → fuente → estrategia → clase compartida
3. Las actividades se agrupan por momento en el orden fijo: apertura → desarrollo → cierre
4. Si la clase es compartida, se incluye el nombre de la otra materia y sus topics
5. Campos vacios/nil se omiten para evitar ruido en el prompt

## Notas

- Este builder es usado exclusivamente por HU-5.4 (GenerateLessonProposal)
- Las actividades se agrupan por momento en el orden fijo: apertura → desarrollo → cierre
- El contexto de clase compartida es critico — le indica al LLM que considere conexiones interdisciplinarias
- Las notas del docente sobre actividades se incluyen porque llevan intencion de personalizacion importante
- El builder recibe un struct de input pre-armado, NO IDs crudos — el usecase es responsable de cargar los datos
