# T-6.5.3: Middleware de feature flags de IA

**Historia:** HU-6.5 — Customizacion por organizacion
**Tipo:** Backend
**Estado:** ⬜ Pendiente
**Fase:** Post-MVP

---

## Descripcion

Implementar un middleware que valide si la feature de IA solicitada esta habilitada para la organizacion del usuario. Si la feature esta deshabilitada, retorna 403 Forbidden antes de llegar al handler.

## Implementacion

```go
// src/infrastructure/middleware/ai_feature_gate.go
package middleware

import (
    "net/http"
    "go-alizia/src/core/entities"
)

// AIFeature identifica una funcionalidad de IA controlable por org.
type AIFeature string

const (
    AIFeatureGeneration      AIFeature = "generation"
    AIFeatureChat            AIFeature = "chat"
    AIFeatureRecommendations AIFeature = "recommendations"
)

// RequireAIFeature retorna un middleware que valida si la feature esta habilitada.
func RequireAIFeature(feature AIFeature) func(http.Handler) http.Handler {
    return func(next http.Handler) http.Handler {
        return http.HandlerFunc(func(w http.ResponseWriter, r *http.Request) {
            org := GetOrgFromContext(r.Context())
            settings := org.GetAISettings() // retorna defaults si no hay config

            if !settings.IsFeatureEnabled(string(feature)) {
                http.Error(w, "AI feature not enabled for this organization", http.StatusForbidden)
                return
            }

            next.ServeHTTP(w, r)
        })
    }
}
```

### Metodo en AISettings

```go
// IsFeatureEnabled verifica si una feature esta en la lista de enabled_features.
func (s AISettings) IsFeatureEnabled(feature string) bool {
    for _, f := range s.EnabledFeatures {
        if f == feature {
            return true
        }
    }
    return false
}
```

### Uso en rutas

```go
// Endpoints de generacion
r.With(middleware.RequireAIFeature(middleware.AIFeatureGeneration)).
    Post("/coordination-documents/{id}/generate", handler.GenerateContent)

// Endpoints de chat
r.With(middleware.RequireAIFeature(middleware.AIFeatureChat)).
    Post("/coordination-documents/{id}/chat", handler.ChatWithAlizia)
```

## Notas

- El middleware se aplica por ruta, no globalmente — cada endpoint declara que feature requiere
- Si `enabled_features` esta vacio en la config, se usan los defaults (`["generation", "chat"]`)
- La respuesta 403 incluye un mensaje claro para que el frontend pueda mostrar un estado apropiado (ej: "Esta funcionalidad no esta disponible para tu institucion")
- La feature `recommendations` se reserva para el comportamiento proactivo de HU-6.6
